# Simple Api
## A simple API using cors and mongoDB based on (Rocketseat)[https://app.rocketseat.com.br/] free Node.js course

## Installation

Use the package manager [yarn](https://yarnpkg.com/getting-started/install) to install dependencies.

```bash
yarn install
```

## Usage

```Bash
yarn dev
```
